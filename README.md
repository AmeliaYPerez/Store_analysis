# Retail Store Performance & Investment Analysis

A complete data science case study analyzing 4 retail stores to identify the best investment opportunity. This project demonstrates end-to-end data analysis skills from data loading and cleaning to visualization and business recommendations.

**Project Highlights:**
- Processed multiple CSV datasets with Pandas
- Created interactive visualizations and dashboards
- Developed weighted scoring system for store evaluation
- Delivered data-backed investment recommendations
- Analyzed key metrics: revenue, ratings, shipping, product performance

**Skills Demonstrated:** Python, Pandas, Matplotlib, Seaborn, Data Analysis, Business Intelligence
